# Attribute: String

<when>Text input, URLs, icon names, alignment values</when>

## block.json Schema

```json
"attributeName": {
	"type": "string",
	"default": "Default text"
}
```

## Common Patterns

### Text Input
```json
"heading": {
	"type": "string",
	"default": ""
}
```

### URL/Link
```json
"ctaUrl": {
	"type": "string",
	"default": "#"
}
```

### Icon (Bootstrap Icons)
```json
"icon": {
	"type": "string",
	"default": "bi-star-fill"
}
```

### Alignment
```json
"textAlign": {
	"type": "string",
	"default": "left"
}
```
